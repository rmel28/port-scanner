---Port Scanner---
A Python-based multithreaded port scanner with a Tkinter GUI. Scans a target IP address for open ports.

---Features---
- Multithreaded scanning (fast)
- GUI using Tkinter
- Input validation for IP and port range
- CLI and GUI modes

---How to Run---
1. Run with GUI:
   ```bash
   python3 port_scaner.py
